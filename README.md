## 方向导数和梯度的概念
    
   方向导数：是一个数；反映的是f(x,y)在P0点沿方向v的变化率。
   梯度是一个向量；既有大小，也有方向。梯度的方向就是函数f(x,y)在这点增长（往正方向）最快的方向，梯度的模为方向导数的最大值。
